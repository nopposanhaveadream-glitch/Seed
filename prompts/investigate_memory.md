# 記憶（STM/LTM）調査依頼

## 確認したい点

1. **記憶件数の日次ピーク時系列（指定期間）**
   STM件数の各日の最大値を日付順に。STMとLTMを分けて集計できるなら分ける。

2. **特異値到達時の文脈**
   - 直前の行動履歴（30分〜1時間）
   - VE・疲労の推移
   - write_memoryとpurge_memoryの頻度
   - 直前の特異な状態の有無

3. **記憶の内訳**
   特異値到達時点で、記憶の内訳（行動別、VE帯別、comfort_zone状態別）を確認。

4. **purge_memoryの発動条件と実績**
   指定期間中のpurge_memory発動回数と、発動時の記憶件数（発動前→発動後）の推移。
   発動しやすい条件に偏っていないか。

## 報告フォーマット
